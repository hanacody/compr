■ rankuplog v2.5소개

rankuplog (이하 rankuplog) v2.5은 간단한 소스코드를 적용할 페이지에 넣는 방법으로
시간,날짜,요일별 접속자수 통계 관리를 할 수 있습니다.
그리고 접속자들의 접속경로를 파악하여 어느곳에 자신의 페이지가 링크되어 있는지를 알아볼 수 있습니다.


■ rankuplog v2.5의 기능

일별 접속자수 표시
총 접속자수 표시
시간별 통계
날짜별 통계
요일별 통계
월별 통계
접속주소 경로 기록
접속전 아이피 기록



■ rankuplog  설치방법

rankuplog를 설치하려면 php와 mySQL이 지원되는 계정이어야 합니다.

1. 다운을 받으신후에 압축을 풉니다.
2. 에디터를 이용하여 dbcon.php 파일에 디비정보를 정확하게 기재합니다
	$connect_host="호스트명";
	$connect_id="DB아이디";
	$connect_pass="DB패스워드";
	$connect_db=" DB이름";

3. 압축파일을 푼 후에  FTP로 접속하여 적당한 폴더를 만드시고 , 폴더 안에 모든 파일을 업로드합니다
4. rankuplog_install.html 를 실행시킵니다.(브라우저에서 실행)
예) http://URL/설치한폴더/rankuplog_install.html

5. DB에 접속하기 위해 관리자아이디/관리자패스워드 을 입력하시면 자동으로 설치됩니다.

6. 설치가 완료된후 로그인페이지로 넘어가게 됩니다.
7. 관리자 로그인은 위에서 입력하신 관리자아이디/관리자패스워드로 하시면 됩니다.

8. 관리자로 로그인하기 위해서는 http://URL/설치한폴더/index.html


■ rankuplog  웹페이지에 적용 방법


카운터를 페이지에 설정하는 방법은 include를 이용하는 것입니다.
적용할 페이지(가급적이면 index페이지에 적용하는게 좋습니다.)의 맨 윗줄에 아래 코드를 적어줍니다.

<?
$newsnad_cook='newsnad';
setCookie('newsnad_id',$newsnad_cook,time()+86400,'/');
if(!$newsnad_id){
include "./설치한폴더/dbcon.php";
include "./설치한폴더/rankuplog.php";
}
?>
위에 소스에서 86400은 24시간을 말하는 겁니다
86400 으로 지정해서 사용하시면 24시간내에 재방문은
1개의 카운터로 적용시킵니다.

만약 30분으로 지정하고 싶으시면
86400 을 (60초x30분)1800 으로 변경하시면 됩니다

위에 소스를 사이트 첫화면(index.html)에 삽입해서사용하시면 됩니다
주의 하실 점은 위에 소스가  페이지 상단에 있어야 하는겁니다

주의) 만약 위의 소스를 삽입하기 전에 웹브라우저에 출력시키는 다른
문구가 있다면 에러가 발생합니다

원 제작자를 표기하지 않은 수정배포를 허용하지 않습니다.
