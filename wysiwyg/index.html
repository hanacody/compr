자바스크립트를 이용한 위지윅(WYSIWYG) 에디터입니다.
웹메일에 쓰려고 만들었는데...
기능은 별로지만.. 그냥 깔끔하고 심플하게 쓰고 싶을때..
사용하면 좋을것 같네요..

사용방법은..
1. 상단에 "wysiwyg.js"를 호출하는 한줄 추가.
2. 에디팅 하고 싶은 TEXTAREA 태그안에 type="editor" 라고 추가해주세요.
3. Editor 의 크기가 textarea에 스타일로 지정된 넓이와 높이를 참조하므로..
아래처럼 스타일로 지정해주세요.
<textarea name="content" type="editor" style="width:700px; height:300px;">
4. wysiwyg.js 파일안에 아래와 같은 곳이 있는데요.. X좌표를 알맞게 설정하시면 됩니다.
var left1 = (navigator.appName == "Microsoft Internet Explorer")?"13px":"9px"; //화면에 따라 약간씩 정해줘야해요..(IE:FireFox)
var left2 = (navigator.appName == "Microsoft Internet Explorer")?"53px":"49px"; //화면에 따라 약간씩 정해줘야해요..(IE:FireFox)
var left3 = (navigator.appName == "Microsoft Internet Explorer")?"179px":"175px"; //화면에 따라 약간씩 정해줘야해요..(IE:FireFox)
var left4 = (navigator.appName == "Microsoft Internet Explorer")?"197px":"193px"; //화면에 따라 약간씩 정해줘야해요..(IE:FireFox)
특징은..
1. 그럴일이야 살면서 몇번이나 있으랴만은.. 한 화면에 복수의 에디터가 필요한 경우가 생기더라도.. 잘 작동합니다.
2. IE전용 인것들이 많아서.. IE와 파이어폭스에서 돌아가도록 했습니다. IE6.0 / FF1.5
